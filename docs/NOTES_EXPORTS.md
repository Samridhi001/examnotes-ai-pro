# Notes Export Flow

ExamNotesAI Pro supports exporting generated notes for offline revision and sharing.

## Export Options

- Copy Markdown: copies a clean study-notes version to clipboard.
- Download MD: saves generated notes as a Markdown file.
- Download JSON: saves the complete note object for debugging or future imports.
- Print / PDF: opens the browser print flow so users can save notes as PDF.

## Why Markdown?

Markdown is lightweight, readable, portable, and easy to convert into PDF, docs, or web content later.

## Why JSON?

JSON keeps the complete generated structure, including summary, questions, flashcards, diagrams, charts, and provider metadata.

## Print View

The print stylesheet hides forms, navigation, history panels, and export controls so the final PDF focuses only on the generated study material.
